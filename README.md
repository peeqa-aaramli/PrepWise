Project Overview

PrepWise is a lightweight, intuitive desktop study planner designed specifically to help students manage their academic workloads. Built entirely on the JavaFX framework, the application separates layout and business logic from visual aesthetics using an externalized CSS architecture.

PrepWise provides students with structured environments to organize tasks by academic subjects, track pressing deadlines, and capture fleeting notes within an interactive calendar matrix.

The files contained are:

1. AssignmentsView.java (/src/com/studyplanner/): Manages the primary assignment hub using a split master-detail panel to track subjects alongside an editable JavaFX TableView.

2. CalendarView.java (/src/com/studyplanner/): Builds the month-view scheduler using an auto-calculating GridPane matrix to handle dates, navigation, and live note-saving.

3. appstyle.css (/src/resources/): The master stylesheet housing the project's purple color palette, layout spacing, button hover states, and typography rules.

4. App.java (/src/com/studyplanner/): Serves as the main entry point that initializes the primary stage, loads the master stylesheet, and navigate between the calendar and assignment  views.
